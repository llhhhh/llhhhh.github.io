git 操作

2019-08-30 1 min read # git

直接克隆项目

git clone https://git.coding.net/***/***.git

已经创建项目要提交到仓库

1. git init
2. git remote add origin  https://git.coding.net/***/***.git
3. git status //查看一下状态
4. git add . //如果是提交全部用点就可以。
5. git push origin master //如果要上传到其他分支，请先创建分支

回退版本

//回退到上个版本
git reset HEAD^
//回退到某个版本
1. git log
2. git reset --hard commit_id
      • 直接克隆项目
      • 已经创建项目要提交到仓库
      • 回退版本
