数组与字符串相互转换

2019-08-30 1 min read # JavaScript

字符串转成数组

var str = 'hello';
str.split('');
console.log(str) //['h','e','l','l','o']

数组转字符串

var str = ['h','e','l','l','o'];
str.join('');
console.log(str) // 'hello'
